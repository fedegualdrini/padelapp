# Code of Conduct

This project is committed to a friendly, respectful, and inclusive environment.

## Our Standards
- Be kind and professional.
- Assume good intent and be constructive.
- Respect differing viewpoints and experiences.
- No harassment, discrimination, or personal attacks.

## Scope
These standards apply in project spaces and when representing the project.

## Reporting
If you experience or witness unacceptable behavior, please report it via GitHub Security Advisories or by contacting the maintainers.

## Enforcement
Project maintainers may remove or edit contributions that violate this Code of Conduct.
