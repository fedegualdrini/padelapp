# Contributing

Thanks for contributing to Padel Tracker!

## Getting Started
1. Fork the repo and create a branch from `main`.
2. Install dependencies: `npm install`
3. Create a `.env.local` file (see README).
4. Run the app: `npm run dev`

## Code Style
- Use TypeScript.
- Keep changes scoped and focused.
- Prefer small, reviewable pull requests.

## Before You Submit
- Run `npm run lint` and `npm run build`.
- Add or update documentation if behavior changes.
- Include screenshots for UI changes.

## Commit Messages
- Use short, descriptive messages (e.g., `fix: handle invalid passphrase`).

## Reporting Issues
- Use the issue templates and include clear repro steps.

## Security Issues
- Please report security issues via GitHub Security Advisories.
